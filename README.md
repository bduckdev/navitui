# navitui

An interactive TUI for navidrome.

## Planned Features

- Interactive TUI at `navitui`
- Simple fuzzy search through library at `navitui select`
- CLI search at `navitui --first "I LIKE THE NOISE (I MADE THIS SONG IN 1 HOUR WHILE POOPING"`
